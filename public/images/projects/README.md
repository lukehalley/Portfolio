# Project Images

Place project logo/icon images in this folder. Images will appear as faded backgrounds on the right side of project cards.

## Required Images

### Can be sourced automatically (logos from public sources):
- `spotify.png` - Spotify logo
- `reddit.png` - Reddit logo
- `crypto.png` - Cryptocurrency/Bitcoin logo
- `haskell.png` - Haskell language logo
- `algorithm.png` - Generic algorithm/code icon
- `audio.png` - Audio waveform/speaker icon
- `robotics.png` - Robot/IoT device icon
- `marketplace.png` - Shopping cart/marketplace icon

### Need manual creation/photos:
- `stork.png` - Stork project logo/icon (your IoT tracking device project)
- `ndis.png` - NDIS Ready logo/icon (your production SaaS)

## Image Specifications
- Format: PNG (for transparency)
- Size: 800x800px recommended (will be scaled)
- Style: Clean, minimal logos work best
- Will be displayed at 5% opacity as background decoration
